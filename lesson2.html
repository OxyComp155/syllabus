Assignment #2: Style Your Autobiographical Page Experiment

Use the autobiographical page that you created last week to experiment with styling in CSS. Remove all the formatting and layout tags before you begin styling. Think about how to create a visually compelling and spatially sensible layout. Think about how the layout interacts with the content of the page. Think about how you would most effectively organize the information, what you want to emphasize on the page, and how your audience will view the page.

Refer to the CSS reference to broaden your CSS vocabulary.
